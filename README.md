# Журнал регистрации УТЗ

Корпоративная система регистрации документов с автоматической нумерацией и резервированием номеров.

---

## Основные возможности

- **Автоматическая нумерация** документов в формате `УТЗ-XXXXXX`
- **"Золотые" номера** (заканчивающиеся на 00) — доступны только администраторам
- **Система резервирования** — безопасное выделение пула номеров пользователям
- **Мастер регистрации** — удобный пошаговый интерфейс
- **Гибкие отчёты** с фильтрацией и экспортом в Excel
- **Административная панель** для поиска и редактирования документов
- **Аудит изменений** всех административных действий

---

**Инфраструктура:** Docker, Docker Compose, Nginx, Alembic

## Использование

### Роли пользователей

Система использует заголовок `X-User` для определения пользователя и его прав:

- **Администраторы** — логины, указанные в переменной `ADMIN_USERS` в файле `.env`
- **Обычные пользователи** — все остальные

Администраторы имеют доступ к:
- Резервированию «золотых» номеров
- Административной панели редактирования
- Расширенным отчётам

### Основные разделы интерфейса

- **`/`** — **Мастер регистрации** (основной рабочий интерфейс)
- **`/reports`** — Отчёты и аналитика с возможностью экспорта в Excel
- **`/admin`** — Административная панель (поиск и редактирование документов)

## Для разработчиков

```bash

# Просмотр логов
```

## Ключевые бизнес-правила

- Документ уникален по комбинации `(doc_name, note, equipment_id)`
- Обычные пользователи не могут резервировать и назначать «золотые» номера
- Неиспользованные номера автоматически освобождаются по истечении TTL
- Все административные изменения фиксируются в журнале аудита
